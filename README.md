# Sistema CRUD de Produtos

Sistema CRUD completo (BackEnd e FrontEnd) desenvolvido em Java para demonstrar competências do ciclo completo de desenvolvimento.

## Destaque

Sistema contempla:

- Boas práticas de programação (Clean Code e Design Patters);
- Cobertura unitária de testes;
- Cobertura de testes da interface (Usando o Selenium);
- Ciclo completo de CD/CI, com coverage e Quality Gate Status (usando o SonarQube).


## Visão Geral

O propósito deste pequeno aplicativo é demonstrar o desenvolvimento o ciclo completo de desenvolvimento de software para a diciplina 25E4_5-PB-EngenhariaDisciplinadaDeSoftwares (Projeto de Bloco) TP4


### Autor

Anderson Machado


## Uso

Quando operacional, o acesso pode ser feito em qualquer Browser usando o seguinte link: http://localhost:8080


## Instalação

- Faça a cópia do repositório usando o seguinte comando: 
  git clone https://github.com/andersonsmachadoinfnet/25E4_5-PB-EngenhariaDisciplinadaDeSoftwares-TP4.git

- Construa a imagem e execute o contêiner docker usando o seguinte comando:
  docker compose up --build

- Abra o browser com o seguinte endereço: http://localhost:8080
